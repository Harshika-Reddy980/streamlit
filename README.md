## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/Harshika-Reddy980/streamlit.git
```

2. Navigate to the project folder:

```
cd streamlit
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
